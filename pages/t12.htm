    Тема 12. Принципы разделения каналов


    рис.12.1. упрощенная структура МСП

    Пропускная способность линии связи может значительно превышать пропускную способность необходимую для организации одного канала. В данной ситуации для повышения эффективности целесообразно использовать многоканальные системы передачи(МСП). Многоканальная система обеспечивает передачу по одной физической цепи большого числа каналов. На рисунке 12.1 показана упрощенная структура МСП.

    Передающая и приемная станции S1…SN – индивидуальные сигналы с помощью модуляторов на передающей станции преобразуются М1…МN в канальные сигналы УО – устройство объединения.

    u(t) – групповой сигнал, который передается по линии передачи, РУ – разделяющее устройство, U1…UN c помощью демодуляторов Д1…ДN преобразуются в информационные каналы S1…SN.

    Чаще всего используют частотный и временной способы разделения каналов (ЧРК и ВРК).

    Частотное разделение каналов

    Один из способов разделения канальных сигналов (или разделения каналов) заключается в следующем. В качестве переносчиков выбирают гармонические несущие колебания с различными частотами. В результате каждый первичный сигнал после преобразования в канальный сигнал (т.е. после модуляции) будет размещаться в своей полосе частот. В качестве примера на рис. 2.3 показано преобразование N первичных сигналов, имеющих одинаковые спектры, путем модуляции по амплитуде (АМ) несущих колебаний с различными частотами. Интервал между несущими частотами соседних каналов должен быть таким, чтобы полосы частот канальных сигналов не перекрывались.


    Рис. 12.2. Многоканальная система передачи с частотным разделением каналов

    На рис. 12.2 представлена структурная схема многоканальной системы передачи. Первичные сигналы s1(t), s2(t), ..., sN(t) преобразуются устройствами М1, М2, ..., МN. модулированные несущие колебания v1(t), v2(t), ..., vN(t), полученные на выходе этих устройств, называются канальными сигналами. В отличие от первичных сигналов, имеющих общий спектр, канальные разнесены по спектру (рис. 12.3). Групповой сигнал v(t) получается объединением канальных сигналов v1(t), v2(t), ..., vN(t) в устройстве объединения (УО).

    Рис. 12.3. Преобразование первичных сигналов в канальные

    Самым распространенным методом формирования канальных сигналов является метод формирования одной боковой.

    Можно передавать одну боковую полосу, так как она уже содержит всю информацию об исходном сигнале.


    При этом методе происходит лишь перемещение сигнала по шкале частот, поэтому этот метод получил название “преобразование частот”

    Преимуществами этого метода являются: узкая полоса частот канального сигнала; так как несущая не передается, энергия тратится лишь на передачу одной боковой полосы.

    Необходимость восстановления несущей на приеме ведет к изменению спектра на величину ∆ω, что ухудшает качество передаваемой информации. Так же недостатком является сложность подавления несущей и второй боковой полосы на передающей станции.

    На приемном конце канальные сигналы выделяются из группового с помощью разделительных частотных фильтров Ф1, Ф2, ..., ФN, пропускающих сигналы своего канала и подавляющих остальные. Восстановление первичных сигналов из канальных производится с помощью демодуляторов Д1, Д2, ..., ДN.

    Временное разделение каналов

    Рис. 12.4. Многоканальная система передачи с временным
    разделением каналов

    В основе лежит теорема Котельникова: любой сигнал, имеющий непрерывный спектр и ограниченный Fв может быть представлен в виде отчетов на передаче и затем восстановлен на приеме со сколь угодно малой погрешностью, если частота отчетов Fд ≥ 2Fв.

    При ВРК по цепи передаются периодические последовательности очень коротких импульсов, амплитуды которых равны величинам мгновенных значений канальных сигналов.

    Исходные непрерывные сигналы каждого канала S1(t)÷SN(t) после ограничения по спектру фильтром нижних частот (ФНЧ) поступают на ключи ЭК1÷ЭКN, которые осуществляют дискретизацию этих сигналов. Работой ключей управляет генераторное оборудование (ГО) на передаче: оно вырабатывает периодическую последовательность импульсов с Fд. Импульсные последовательности, управляющие работой ключей, сдвинуты друг от друга во времени на равные интервалы. Величина этих интервалов определяется временем дискретизации и количеством каналов. Таким образом, групповой сигнал представляет собой последовательность коротких импульсов.

    Рис. 12.5. Временной принцип объединения канальных
    сигналов (а–в) в групповой (г)

    На приемной стороне разделение исходных сигналов осуществляется с помощью канальных ключей ЭК1÷ЭКN. Работой ключей управляет последовательность импульсов от ГО на приеме. Для того, чтобы передоваемый сигнал поступил в соответствующий приемник необходимо, чтобы электронные ключи на приеме и передаче работали синхронно и синфазно. Для этого с передающей стороны на приемную передается синхросигнал. В том случае, если ключи будут работать не синфазно сигнал попадет не в тот приемник, а в приемник соседнего канала. При несинхронной работе ключей сигнал может вообще ни попасть ни в один из приемников.

    Восстановление исходного сигнала на приеме осуществляется с помощью ФНЧ. Так как полоса пропускания ограниченна, то между каналами в групповом сигнале могут возникнуть взаимные переходные помехи. Для их уменьшения между канальными импульсами вводят защитный интервал T.
